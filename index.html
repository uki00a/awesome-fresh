Modules

  • fresh_charts - A server side rendered charting library for Fresh based on Chart.js.
  • fresh-session - Dead simple cookie-based session for Deno Fresh.
  • fresh-seo - The fastest way ⚡ to create sitemap and robots.txt in your Deno Fresh project :lemon:
  • fresh-tailwindcss - Fresh plugin for TailwindCSS
  • fresh-testing-library - Utilities for testing fresh apps.
  • TSX Tabler Icons - Easy to use Tabler Icons in your Fresh app. Just import the icons.
  • react-icons - React icons for deno Fresh contains more than 30 000 SVG.
  • Deno KV Insights - Deno KV Insights is a tool for managing Deno KV database entries.
  • Deno KV OAuth - High-level OAuth 2.0 powered by Deno KV.

Frameworks

The following frameworks provide Fresh integration.

  • Danet - A Savory web framework for Deno heavily inspired by Nest
  • Remult - Build Full-stack, End-to-end Type-safe CRUD Apps without the Boilerplate

Templates

  • Deno SaaSKit - A modern SaaS template built on Fresh.
  • Deno Fresh OpenAI Doc Search Starter - Template for building your own custom ChatGPT style doc search powered by Fresh, Deno, OpenAI, and Supabase.
  • Fresh WordPress Themes - This repository includes Fresh WordPress themes. You can start developing WordPress-backed Fresh website using these templates.

Apps

  • Deco - Open-Source web editor based on Preact, Tailwind and TypeScript. The other side of code.
  • jsr.io - The open-source package registry for modern JavaScript and TypeScript.
  • kview - A web interface for viewing Deno KV stores.
  • Netzo - Build business web apps fast, with less resources with an opinionated fresh meta-framework and development platform.
